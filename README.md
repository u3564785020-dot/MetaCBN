# Meta Pay - Meta

Meta Pay imitation page with support chat.

## Local Development

```bash
npm install
npm start
```

Server will run on http://localhost:3000

## Deployment on Railway

This project is configured for deployment on Railway.com. Railway will automatically:
- Detect Node.js project
- Install dependencies
- Start the server using `npm start`

The server will use the PORT environment variable provided by Railway.
